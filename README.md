# Car Rental System

A console-based Car Rental System developed to manage car rentals efficiently.  
This project allows users to rent cars, return cars, and manage available vehicles in the system.

---

## Features

- View available cars
- Rent a car
- Return a car
- Customer management
- Rental price calculation
- Input validation
- Simple and user-friendly interface

## How It Works

1. The system stores a list of available cars.
2. Customers can select a car to rent.
3. Rental price is calculated based on the number of rental days.
4. When a car is returned, it becomes available again.
5. The system updates records automatically.
